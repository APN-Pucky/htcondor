%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\subsection{Installing PVM Support in Condor}
\label{sec:Install-PVM-Condor}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

To install support for PVM in Condor, download the file archive from
\Url{http://www.cs.wisc.edu/condor/condor-pvm} and follow the directions
found the \File{INSTALL} file contained in the archive.
